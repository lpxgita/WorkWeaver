# 1.0.8（2026-02-11）

## 主要变更
- Electron 界面左下角侧边栏底部新增版本号显示
- 版本号通过 IPC（`app:version`）从主进程动态获取，与 package.json 同步
- 标题栏和页面标题统一更新为 WorkWeaver

## 变更文件
- `electron-app/renderer/index.html`：侧边栏底部添加 `.sidebar-version` 元素，标题更新为 WorkWeaver
- `electron-app/renderer/styles.css`：新增 `.sidebar-version` 样式（底部对齐、等宽字体、淡灰色）
- `electron-app/preload.js`：新增 `getVersion` API 暴露
- `electron-app/main.js`：注册 `app:version` IPC handler
- `electron-app/renderer/app.js`：init() 中动态加载版本号
- `electron-app/package.json`：版本号 1.0.7 → 1.0.8
