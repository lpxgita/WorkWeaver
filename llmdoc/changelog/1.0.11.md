# 1.0.11（2026-02-12）

## 主要变更
- 行为目录引入三分类：用户主动设置行为、AI 提出且最近 7 天有分类记录行为、AI 提出且最近 7 天无分类记录行为
- Prompt 组合逻辑调整：仅注入前两类行为，自动排除“AI 提出但最近 7 天无分类记录”的陈旧行为
- 新增行为近期使用判定：从最近 7 天 `2min` 总结中提取行为分类记录作为保留依据
- 用户在 Todo 页手动新建行为时，写入 `source: "user"`，与 AI 自动创建行为的 `source: "ai"` 形成统一来源标记

## 变更文件
- `ai_summary/src/prompt-builder.js`：新增行为三分类、最近 7 天分类记录扫描、行为目录过滤与缓存逻辑
- `ai_summary/main.js`：初始化 `PromptBuilder` 时传入 `summaryDir` 和 `behaviorRecentDays`
- `electron-app/todo-store.js`：`createBehavior()` 新增 `source: "user"` 字段
- `llmdoc/architecture/ai-summary-architecture.md`：补充行为目录三分类过滤说明
- `electron-app/package.json`：版本号 `1.0.10` → `1.0.11`
- `electron-app/renderer/index.html`：侧边栏版本号 `v1.0.10` → `v1.0.11`
