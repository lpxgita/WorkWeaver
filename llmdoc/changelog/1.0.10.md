# 1.0.10（2026-02-11）

## 主要变更
- 时间线样式优化：减小活动条和泳道行的圆角（12px/6px → 3px），边界更清晰
- 修复泳道背景不对齐问题：泳道行宽度显式设为时间轴内容宽度，确保背景填满到最右侧时间刻度
- 新增泳道任务名浮动吸附功能：横向滚动时任务名始终吸附在泳道左侧可见区域，不再被滚出视野
- 默认打开时间线时自动滚动到最右侧，用户直接看到最近的活动

## 变更文件
- `electron-app/renderer/styles.css`：调整 `.summary-timeline-bar` 和 `.summary-timeline-row` 圆角为 3px；移除 `.summary-timeline-canvas` 的 padding-right 改为 JS 动态计算；新增 `.summary-timeline-sticky-label` 浮动吸附标签样式；`.summary-timeline-body` 新增 width: 100%
- `electron-app/renderer/app.js`：`_drawTimeline()` 中重构 canvas 宽度计算（内容宽度 + 100px 预留）、body 显式宽度设置、标签从 bar 内部移至 row 级别浮动元素、新增滚动监听更新吸附标签位置、默认 scrollLeft 滚动到最右侧
- `electron-app/package.json`：版本号 1.0.9 → 1.0.10
- `electron-app/renderer/index.html`：版本号 v1.0.9 → v1.0.10
