# 1.0.9（2026-02-11）

## 主要变更
- 重新设计 Prompt 中 Todo List 信息的组合形式，从纯文本缩进格式改为 XML 结构化标签
  - 任务目录使用 `<task_directory>` / `<task>` / `<active_subtasks>` 等标签，包含任务状态、子任务统计、上下文描述
  - 行为目录使用 `<behavior_directory>` / `<behavior>` 标签，包含行为描述
  - 归类规则使用 `<classification_rules>` / `<aggregation_rules>` 标签，按优先级排列
  - 已完成任务以 `<completed_tasks>` 形式保留历史上下文
- 新增仪表盘一键启停全部服务功能（截图+AI总结），支持状态同步显示
- README 新增 API 费用说明（每小时约 0.3-0.4 美元 Gemini API）
- README 新增隐私风险提示（屏幕截图上传、焦点窗口信息收集）

## 变更文件
- `ai_summary/src/prompt-builder.js`：重构 `_buildTodoContextText()` 为 XML 结构化格式；新增 `_escapeXml()` 工具方法；更新 build2min/build10min/build1h 中的 Todo 注入模板
- `electron-app/service-manager.js`：新增 `startAll()` 和 `isAllRunning()` 方法
- `electron-app/main.js`：新增 `service:start-all` / `service:stop-all` IPC handler
- `electron-app/preload.js`：新增 `startAllServices` / `stopAllServices` API
- `electron-app/renderer/index.html`：仪表盘新增一键启停按钮区域
- `electron-app/renderer/app.js`：新增 `toggleAllServices()` 方法，updateStatusUI 同步一键按钮状态
- `electron-app/renderer/styles.css`：新增 `.all-services-bar` / `.btn-all-start` / `.btn-all-stop` 样式
- `README.md`：新增费用说明和隐私风险提示章节
- `electron-app/package.json`：版本号 1.0.8 → 1.0.9
